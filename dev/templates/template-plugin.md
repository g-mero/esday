## Examples
```typescript
import { esday } from 'esday'
import myNewPlugin from 'esday/plugins/myNew'

esday.extend(myNewPlugin)

esday('08-2023-14 21:43:12.123').myPlugin()
// Returns '24'

esday('08/2023/14 21 43.12 123').myPlugin('24')
// Returns esday for '2024-06-10'
```
